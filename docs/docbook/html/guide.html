1.2. 产品特性

1) 类Todo list产品,并引入GTD概念,用于管理个人时间.

2) 在传统的单纯规划时间的基础上,提供记录功能.即该管理时间理念为:

(1)首先通过便签来列出所需完成的任务(传统Todo list功能)

(2)然后按紧急或重要程度去规划自己的时间(可运用GTD分类或“四象限”法)

(3)最后可在进行每一项任务的开始和结束时记录它们的时间(即跟踪计划的成效).

(4)事后便可通过日志数据来总结自己时间管理的成效(对比计划和现实的差异,总结完成任务的效率,记录每天做成了哪些任务)

2.1. 便条操作

2.1.1. 添加

用户可在主输入框内输入字符串,将其解析并发送给后台服务器,存入数据库.

输入的内容可包含:便签内容,到期时间,重要级别,TAG和分类.

2.1.2. 删除

通过点击[删除]按钮,删除便签.

2.1.3. 修改

直接单击便签文字区域便可修改便签内容,若需修改其他详情(除分类外),则需要点击[修改详情]按钮.

2.1.4. 排序

直接拖拽可手动排序便签,也可通过[到期时间],[重要级别]等自动排序.

2.1.5. 分类

直接拖拽便签到指定分类.

2.2. 分类操作

2.2.1. 添加

在[分类管理]页面,新建分类,

也可直接点击[添加分类]按钮新建分类

2.2.2. 删除

在[分类管理]页面,删除分类

2.2.3. 重命名

在[分类管理]页面,重命名分类

2.3. 个人资料操作

2.3.1. 注册

通过[注册]按钮进入注册页面(可修改为在当前页面弹出注册/登录框)

2.3.2. 修改

在[个人资料]页面可修改详细个人资料

2.3.3. 注销

在[个人资料]页面可点击[注销帐号]

注意:注销只冻结帐号,并不自动删除帐号及资料.管理员可手动删除.

2.4. 登录功能

检查用户名/用户密码/验证码.

提供[记住我]功能,一个月之内在该客户端上无需再次登录.
